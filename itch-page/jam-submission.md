# GMTK Game Jam 2026 Submission

## How does the game fit the theme "COUNT DOWN"?

Every attempt is a literal ten-second countdown. When it reaches zero, the run becomes a ghost that repeats the player's actions while the countdown starts again. Progress comes from arranging those short, expiring timelines so past selves can cooperate with the current one.
